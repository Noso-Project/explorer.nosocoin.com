API Calls


Return data from Nosod (nosonode)

  • Returns current block height
    http://api.nosocoin.com:3000/blockheight
    https://api.nosocoin.com:3443/blockheight
  • Returns total supply
    http://api.nosocoin.com:3000/totalsupply
    https://api.nosocoin.com:3443/totalsupply
  • Returns current Noso supply
    http://api.nosocoin.com:3000/circulatingsupply
    https://api.nosocoin.com:3443/circulatingsupply
    https://api.nosocoin.com:3443/circulatingsupply2
  • Returns remaining amount of Noso to be distributed
    http://api.nosocoin.com:3000/undistributedsupply
    https://api.nosocoin.com:3443/undistributedsupply
  • Returns pending transactions on current block
    http://api.nosocoin.com:3000/pendingtx
    https://api.nosocoin.com:3443/pendingtx
